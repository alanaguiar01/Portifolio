O que é?

Front-End é a camada do seu site no qual o usuário interage. É a interface navegável. A fachada. O layout após desenhado, vem como imagem e consequentemente com todos os estilos, medidas especificações especificações técnicas definidas.

O Desenvolvedor Front-End tem a responsabilidade de traduzir isso em código, usando as mais avançadas tecnologias para que os requisitos sejam fielmente seguidos.

Tecnologias utilizadas:

  • HTML (Linguagem de Marcação de Hipertexto);
  • CSS (Folhas de Estilo em Cascatas);
  • JavaScript (linguagem de programação que funciona do lado cliente);
  • Newsletter / Email Marketing.
